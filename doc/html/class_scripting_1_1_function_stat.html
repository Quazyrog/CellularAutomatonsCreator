bool Scripting::FunctionStat::isVolatile ( ) const
overridevirtual

The isVolatile is used to determine if function's value returned from constant formula is constant. For example this class' function valueOf(arg) retruns 2*arg+1. Consequently foo(3) always will be equal 7 – it is not volatile. In contrary function Scripting::FunctionOffset's result depends on cell position selected bay automaton. Therefore OFFSET(-1;-1) (state of up-left neighbour of selected cell) will return differtnt values even if it has constant arguments – it is volatile. This will not let Calculator treat it as constant value during optimization.

Returns
true if and only if function is volatile. In such case returns false

Calculator::optimize
