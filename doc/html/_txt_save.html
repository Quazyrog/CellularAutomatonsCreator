Automaton txt save format

This page describes accepted format of contents of file witch can be opened as cellular automaton.

Introduction

Cellular automatone used by this programs can be read and saved to plain text files. They are divided into three sections (described as 'Section I', 'Section II' and 'Section III'). First of them describes set of states accepted by automaton, After it follows a section describing automatons rule and the last one contains informations about grid. The first line of file is not a part of any section. It contains automaton's name (can contain spaces).
