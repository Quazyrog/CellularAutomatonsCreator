Detailed Description

The CellularAutomaton class represents cellular automaton. It contains information about the grid, states available in the automaton and it's rules. The type used for representing cell's state is quint16.

Member Enumeration Documentation

enum Scripting::CellularAutomaton::Instruction : char
strongprivate

Instructions used to execute next generation function. They determine how should executed ScriptLine be treated. When automaton reads script, it translates all lines of script into ScriptLine structures, and this enum represents all general cases of script lines.

Enumerator
RETURN_STATE 

Equivalent of STATE ...</state>

COMPARATOR_EQUAL 

Equivalent of IF ... = ... THEN ... ELSE ...</state>

COMPARATOR_DIFFERENT 

Equivalent of IF ... <> ... THEN ... ELSE ...</state>

COMPARATOR_GREATER 

Equivalent of IF ... > ... THEN ... ELSE ...</state>

COMPARATOR_GREATEER_EQUAL 

Equivalent of IF ... >= ... THEN ... ELSE ...</state>

COMPARATOR_LESS 

Equivalent of IF ... < ... THEN ... ELSE ...</state>

COMPARATOR_LESS_EQUAL 

Equivalent of IF ... <= ... THEN ... ELSE ...</state>
