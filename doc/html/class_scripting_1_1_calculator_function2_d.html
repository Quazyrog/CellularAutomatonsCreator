Detailed Description

The CalculatorFunction2D class is basic class for calculator's functions with one parameter. Pointer to CalculatorFunction2D may be instelled in calculator. Then it can be interpreted in formulas using syntax name(expr), where name is function's name and expr is an formulas witch value is given to function CalculatorFunction2D::valueOf. For example this class represents function called foo, witch for given number x returns number 2*x+1. Exemplary formula that uses it would be foo(3+foo(0))+1.

To create another function you need to create a subclass of this one and reimplement virtual functions.

bool Scripting::CalculatorFunction2D::isVolatile ( ) const
virtual

The isVolatile is used to determine if function's value returned from constant formula is constant. For example this class' function valueOf(arg) retruns 2*arg+1. Consequently foo(3) always will be equal 7 – it is not volatile. In contrary function Scripting::FunctionOffset's result depends on cell position selected bay automaton. Therefore OFFSET(-1;-1) (state of up-left neighbour of selected cell) will return differtnt values even if it has constant arguments – it is volatile. This will not let Calculator treat it as constant value during optimization.

Returns
true if and only if function is volatile. In such case returns false
